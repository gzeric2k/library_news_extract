================================================================================
NewsBank AI智能下载器 - 快速参考
================================================================================

【交互式模式】(推荐)
  python newsbank_ai_downloader.py
  
  流程:
  1. 输入 URL
  2. 查看 URL 解析结果
  3. 查看参数确认界面
  4. 输入 y 确认执行

【命令行模式】(兼容)
  python newsbank_ai_downloader.py "URL"
  
  仍然显示参数确认界面，输入 y 确认

【跳过确认】(自动化)
  python newsbank_ai_downloader.py "URL" --skip-confirm
  
  直接执行，不显示确认界面

================================================================================
常用参数
================================================================================

AI 筛选:
  --use-llm              使用 LLM 智能判断
  --use-bert             使用 BERT 语义筛选
  --threshold 0.3        相关性阈值 (默认: 0.4)

下载配置:
  --max-pages 5          最大扫描页数 (默认: 10)
  --download-limit 100   最大下载数量 (默认: 50)
  --headless             无头模式

其他:
  --skip-confirm         跳过参数确认
  --interactive          下载前确认

================================================================================
使用示例
================================================================================

1. 基础交互式:
   python newsbank_ai_downloader.py

2. 使用 LLM:
   python newsbank_ai_downloader.py --use-llm

3. 调整阈值:
   python newsbank_ai_downloader.py --threshold 0.3

4. 直接指定 URL:
   python newsbank_ai_downloader.py "https://infoweb-newsbank-com..."

5. 跳过确认:
   python newsbank_ai_downloader.py "URL" --skip-confirm

6. 只扫描不下载:
   python newsbank_ai_downloader.py --max-pages 1

================================================================================
交互式输入说明
================================================================================

URL 输入:
  输入有效的 NewsBank 搜索 URL
  输入 'q' 或 'quit' 退出

参数确认:
  输入 'y' 或 'yes' 确认执行
  输入 'n' 或 'no' 取消执行
  输入 'q' 或 'quit' 取消执行
